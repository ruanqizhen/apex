# Leviathan: Apex (深海巨噬)

一款基于 Web 的 2D 俯视角海洋生存吞噬游戏 (Roguelite 进化 + 弱竞技微操)。玩家从浮游细胞起步，通过不断吞噬更小的海底生物来壮大自身质量，并在升级时选择突变基因进行进化，同时需要躲避体型比自己庞大的顶级掠食者。

## 技术栈

*   **核心引擎**: Vite + React 18 + TypeScript (Strict Mode)
*   **状态管理**: Zustand (Vanilla 状态引擎，与 React 渲染完全解耦)
*   **物理与渲染**: HTML5 Canvas 2D 程序化渲染 (零外部图片/音频资源)

## 核心设计规范

1.  **无头状态模拟 (Headless Engine)**: 游戏的所有物理坐标、速度、AI 决策、碰撞与吞噬计算均在 `src/engine/` 下独立执行，完全不依赖 React 生命周期。React 仅作为只读快照的订阅层渲染 HUD 和进化菜单。
2.  **固定时间步更新 (Fixed-Step Loop)**: 采用 60Hz 固定频率物理逻辑更新与 `requestAnimationFrame` 自适应帧率渲染，在标签切后台时自动进行时间夹持以防止瞬移或“死亡螺旋”。
3.  **空间哈希加速 (Spatial Hash Grid)**: 动态空间哈希网格避免了传统的 $O(n^2)$ 遍历碰撞检测，支持同屏 300+ 实体稳定 60 FPS 游玩。
4.  **对象池策略 (Object Pooling)**: AI 实体和渲染粒子排放均使用复用对象池，杜绝游戏中的垃圾回收 (GC) 卡顿。

## 游戏操作

*   **游动**: 鼠标在屏幕上移动，小鱼会自动平滑地游向鼠标指针。
*   **冲刺加速**: 按住 **鼠标左键** 或 **空格键** 触发冲刺。冲刺会提升 1.8 倍速度，但每秒会消耗当前总质量的 2% (转化为身后的气泡尾迹)。
*   **进化选择**: 当质量达到下一阶段指数阈值时，游戏暂停并弹出 3 张随机突变基因卡片，点击即可获得特定属性强化（支持多层叠加，包含抵挡一致命伤的骨盾、增大判定半径的巨口等）。
*   **狂热连击**: 连续进食增加 Combo 计数，叠满 15 层可获得 5 秒的狂热 Buff (2 倍移速、1.5 倍吞噬半径、免除冲刺质量消耗)。

## 开发命令说明

### 1. 安装依赖

```bash
npm install
```

### 2. 运行本地开发服务

```bash
npm run dev
```

### 3. 项目编译构建

```bash
npm run build
```

编译生成的产品包将输出在 `dist/` 文件夹下。
